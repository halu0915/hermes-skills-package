# 個股深度量能與時間點分析 — [股票代號]

幫我針對 [股票代號] 做完整的技術面 + 基本面分析，依照以下 6 個維度依序產出：

## 1️⃣ 公司定位與產業循環
- 公司主要業務：製造/代理/通路/設計
- 主要代理品牌或客戶（如記憶體通路商→三星、美光）
- 營收比重與地區分布
- 所屬產業循環階段（擴張/成熟/收縮）
- 近期題材催化劑（HBM、AI 伺服器、訂單能見度）

## 2️⃣ 量能累積分析
使用 FinMind API（或 Yahoo Finance）抓取近 6-12 個月日K 資料，分析：
- 近期最大單日成交量（含日期、價格）
- 歷史高點日期與當日量能
- 主升段累積量（從起漲到高點）vs 整理期累積量（從高點到今日）
- 目前累積量是否已「突破歷史前高量能」— 這是關鍵訊號
- 近 5 日、20 日均量

資料來源：
```python
from FinMind.data import DataLoader
api = DataLoader()
api.login_by_token(api_token="...")
df = api.taiwan_stock_daily(stock_id="[股票代號]", start_date="2025-10-01", end_date="today")
```

## 3️⃣ 關鍵價位地圖
列出由近到遠的壓力支撐：
- 今日盤中高低點（即時）
- 近期突破點（如本週跳空缺口）
- 近 30 日高量攻擊點
- 近 3 個月歷史高點
- 終極壓力（半年~一年高點）

每個價位標註：
- 當日量能
- 與現價距離（%）
- 籌碼堆積量（該價位區間累積成交）

## 4️⃣ 主力攻擊時段規律
使用 Yahoo Finance 30分K 資料（近 7-15 日）：
```
https://query1.finance.yahoo.com/v8/finance/chart/[股票代號].TW?interval=30m&range=15d
```

分析：
- 各時段（09:00、09:30、10:00... 13:00）平均量佔比
- 找出主力偏好的攻擊時段
- 偵測「真實攻擊」vs「假突破」：
  - 09:00 大量收黑 = 誘多/出貨
  - 10:00-11:00 放量收紅 = 主力真進場
  - 13:00 尾盤量出 = 隔日續攻訊號
- 列出近 7 日每日最大量時段 + 當時價格變化
- 找出「量>5000張 且 收紅」的關鍵攻擊 K

產出格式：
```
時段   平均量   佔比   判斷
09:00  X,XXX   XX%   開盤測試
09:30  X,XXX   XX%   
10:00  X,XXX   XX%   主力偏好
...
```

## 6️⃣ 時間點預測
依照技術面 + 題材面推估：

**第 1 波啟動（短線 1-2 週）**
- 觸發條件：突破哪個價位 + 量達多少
- 目標價：第一個壓力位
- 時間窗口：具體日期範圍

**主升段啟動（中期 1-2 月）**
- 需要題材：哪些催化劑
- 量能門檻：單日需放大到多少
- 目標價：歷史高點或新高
- 時間窗口：基於歷史整理週期（通常 20-30 天）

**反向訊號（停損預警）**
- 價位：跌破哪個支撐
- 量能：縮到多少視為失敗
- 時間：若 X 天內未突破則轉空

- 📅 重點觀察時間節點（財報、法說、產業數據發布日）

## 格式要求
- 使用表格呈現數據
- 量能單位用「張」（1張 = 1,000 股）
- 百分比保留 2 位小數
- 金額用千分位
- 結論先行（摘要 3 句話）+ 細節佐證
- 加入 Mermaid 圖表視覺化趨勢（如時間線、量能分佈）
