# 服務建議書編寫框架與評委攻略

> 本文總結恩加斯達團隊協助岳暘營造「中壢國小藝文生活通學步道改善工程」標案服務建議書編寫過程中，歸納出的邏輯框架、內容重點、細節劃分及評委偏好預測。

---

## 一、核心原則

### 1.1 評分導向寫作

**不是寫給自己看，是寫給評委打分的。**

每一段文字都要能對應到評選表上的某個評分子項。無法對應的內容 = 浪費頁數。

### 1.2 頁數即資源

服務建議書通常有頁數上限（本案 80 頁）。超過扣分：
- 超 1-10 頁：扣 1 分
- 超 10 頁以上：扣 2 分

**資源分配原則：頁數按配分比例分配**

| 章節 | 配分 | 建議頁數（80頁內）|
|------|------|----------------|
| 第一章 團隊組織 | 10 分 | 8-10 頁 |
| 第二章 工程管理 | 35 分 | 28-30 頁 |
| 第三章 造價分析 | 20 分 | 12-15 頁 |
| 第四章 簡報答詢 | 10 分 | 2-3 頁（簡報另備）|
| 第五章 履約績效 | 25 分 | 18-22 頁 |

### 1.3 差異化三要素

評委一天看 3-5 家廠商的建議書，內容大量重複。脫穎而出靠：

1. **圖說掌握度** — 證明你真的讀過圖、去過現場
2. **數據具體性** — 不說「完善的品管」，說「9 項施工 + 9 項材料自主檢查」
3. **問題預見力** — 主動提出施工困難和對策，而非等評委問

## 二、各章編寫框架

### 第一章：執行團隊組織（10 分）

**評委看什麼：**
- 組織架構是否合理、職責是否清晰
- 關鍵人員的學歷、證照、經驗是否匹配
- 人員是否真的在職（勞保/健保證明）

**編寫要點：**

```
1. 組織架構圖（一頁，清楚呈現指揮鏈）
2. 人員名冊表（表格，含職稱/姓名/學歷/經驗/證照編號）
3. 各職位職掌說明（精簡，每人 3-5 行）
4. 團隊優勢（2-3 點差異化亮點）
5. 協力廠商清單（展示資源調度能力）
```

**常見錯誤：**
- 花大量篇幅寫公司部門職掌（評委不在乎你有幾個部門）
- 職掌描述用通用套話（每家都一樣，沒有鑑別度）
- 缺少證照影本（附件必備）

**加分技巧：**
- 強調關鍵人員的「類似工程」經驗（步道/景觀/校園）
- 標示計畫主持人曾服務的知名營造廠
- 全員在職 = 溝通默契佳（比臨時借牌有說服力）

### 第二章：工程管理（35 分）— 最重要

**評委看什麼：**
- 你是否真的理解這個工程在做什麼
- 施工方法是否具體可行
- 有沒有預見施工困難並提出對策
- 進度規劃是否合理
- 品質/安全/環保管理是否完整

**編寫框架（建議子節）：**

```
2.1 工程概要與認知
    — 工程名稱/地點/業主/金額/工期
    — 主要工項清單（從標單提取）
    
2.2 現場踏勘分析 ★差異化關鍵
    — 衛星圖/街景照片
    — 周邊環境分析（學校/住宅/交通）
    — 施工困難識別 + 對策

2.3 施工進度規劃
    — 甘特圖或網圖
    — 關鍵里程碑表
    — 分階段施工策略

2.4 施工方法
    — 逐項工法說明（對應標單工項）
    — 每段文字後標注引用圖說和標單項次
    
2.5 材料規格與廠牌
    — 主要材料三家以上廠牌對照表
    — 優先採用廠牌及理由
    — 驗苗場（植栽案）

2.6 工程介面整合
    — 內部介面（各工種協調）
    — 外部介面（學校/居民/交通）

2.7 工期評估與縮短對策
    — 預製工法
    — 分區施工
    — 假日/夜間施工

2.8 圖說研判與材料數量統計 ★★核心差異化
    — 逐套圖說分析（L1~LD 每套列出研判重點）
    — 圖說與標單交叉對照表
    — 證明你讀過每一張圖

2.9 品質管理計畫
    — 三級品管制度說明
    — 自主檢查項目清單（施工+材料）
    — 材料送審管制表

2.10 安全衛生管理
    — 校園施工安全特殊措施
    — 上下學時段管制
    — 交通維持計畫

2.11 環保管理
    — 營建廢棄物處理
    — 噪音/揚塵管制
    — 土石方處理
```

**關鍵技巧 — 引用標注：**

每提到一個工項，後面加標注：

> 「混凝土澆置採 210kg/cm²（詳圖說 L3 高程圖，標單壹.二(一)7，數量 522 m³）預拌混凝土...」

這讓評委知道你：讀過圖 → 核對過標單 → 數量正確。三重保證。

### 第三章：造價分析（20 分）

**評委看什麼：**
- 報價是否合理（不是最低價，是合理價）
- 成本結構是否清楚
- 有沒有考慮後續維護

**編寫要點：**

```
3.1 報價金額（須同標單）
3.2 經費組成表
    — 直接工程費（材料+人工+機具）
    — 間接工程費（保險+管理+利潤）
    — 安衛/品管/環保費用
    — 稅捐
3.3 單價合理性說明
3.4 風險預備金
3.5 後續維護計畫（保固期服務承諾）
    — 每季巡檢
    — 透水磚清洗
    — LED 維修時效
    — 植栽養護
3.6 精實作為（成本節省但不減品質的方法）
```

### 第四章：簡報及答詢（10 分）

**服務建議書中只需放：**
- 簡報大綱（5 段時間分配）
- 預擬 Q&A（5-8 題，展示你預見了評委的問題）

**簡報本身另外準備（Gamma/PPT）。**

**預擬 Q&A 重點題目：**
1. 如何確保學童上下學安全？
2. 施工噪音對教學的影響如何降低？
3. 雨季施工如何確保鋪面品質？
4. 為何選用這個廠牌的透水磚？
5. 如何處理居民抗議或施工糾紛？
6. 工期能否縮短？風險在哪裡？
7. 你們做過哪些類似的步道/景觀工程？

### 第五章：履約績效（25 分）

**評委看什麼：**
- 近 5 年有沒有類似工程經驗
- 施工查核成績好不好
- 有沒有職災/停工/逾期紀錄

**編寫要點：**

```
5.1 工程實績表（表格，含工程名稱/業主/金額/期程/類型）
5.2 代表作詳述（2-3 案，含施工照片/成果說明）
5.3 類似工程對照分析（過去經驗 → 本案應用）
5.4 施工查核成績表
5.5 獲獎紀錄
5.6 職災紀錄（0 件 = 重要加分）
5.7 履約情形（無逾期 = 重要加分）
5.8 目前在手工程（證明有產能但不過載）
```

**加分技巧：**
- 做一張「過去經驗 vs 本案需求對照表」，證明每項需求都有對應經驗
- 照片要挑施工中 + 完工後對比，展示品質
- 同一業主的重複案件 = 業主信任的證據

## 三、評委心理分析

### 3.1 評委組成

最有利標評選委員通常包含：
- 學者專家（2 人以上，偏重技術面）
- 機關代表（偏重行政配合度和履約紀錄）
- 可能有使用單位代表（本案可能有校方代表，偏重安全和施工影響）

### 3.2 評委偏好預測

| 評委類型 | 關注重點 | 加分要素 |
|---------|---------|---------|
| 土木/結構學者 | 工法是否正確、材料是否合適 | 具體工法描述、CNS 規範引用 |
| 景觀/都市設計學者 | 設計意圖理解、美觀品質 | 圖說研判能力、材料質感 |
| 機關承辦人 | 配合度、文件完整性、是否按規定格式 | 格式完美、表格清楚、份數足夠 |
| 校方代表 | 學童安全、施工影響最小化 | 安全管理措施、上下學管制 |
| 職安委員 | 安全衛生管理 | 零職災紀錄、安衛計畫詳實 |

### 3.3 扣分雷區

- 服務建議書超過 80 頁（自動扣分）
- 份數不足 12 份（扣 1 分）
- 格式裝訂不符規定
- 簡報非計畫主持人親自簡報（可酌減分）
- 內容與投標文件不一致
- 承諾無法兌現的事項

### 3.4 隱性加分要素

1. **視覺整潔度** — 表格對齊、字體統一、頁碼連續
2. **數據引用** — 每個數字都有出處（圖說/標單/規範）
3. **問題預見** — 主動提出困難比被問到好
4. **在地了解** — 提到里長名字、學校作息、當地交通特性
5. **誠實謙遜** — 不過度承諾，承認限制但提出對策

## 四、編排設計原則

### 4.1 文件結構

```
封面 → 目錄 → 內容摘要暨頁次對照表 → 
第一章 → 第二章 → 第三章 → 第四章 → 第五章 → 
參考資料 → 附件（另冊）
```

### 4.2 表格優先原則

**能用表格就不用段落。** 評委掃讀速度：

- 表格：3 秒看完一頁
- 段落：30 秒看完一頁

關鍵資訊務必用表格呈現：人員名冊、材料廠牌、施工進度、自主檢查項目、實績清單。

### 4.3 引用標注格式

```
「...高壓透水磚鋪設（詳圖說 L5 鋪面系統圖，標單壹.二項，依 CNS 14602 規範）...」
```

三層引用：圖說 → 標單 → 規範。缺一不可。

### 4.4 頁面配置

- A4 直式，雙面印刷
- 圖樣可用 A3 摺頁
- 左側裝訂
- 連續編列頁碼
- 頁眉：工程名稱 + 章節
- 頁腳：公司名稱 + 頁碼

## 五、簡報製作要點

### 5.1 簡報時間分配（15-20 分鐘）

| 段落 | 時間 | 內容 |
|------|------|------|
| 開場 | 1 分鐘 | 公司簡介、團隊介紹 |
| 工程認知 | 5 分鐘 | 圖說研判、數量統計、施工困難 |
| 施工規劃 | 5 分鐘 | 流程、品質、安全 |
| 造價與維護 | 3 分鐘 | 成本結構、保固服務 |
| 實績展示 | 3 分鐘 | 類似工程照片、零職災 |
| 結語 | 1 分鐘 | 承諾、感謝 |

### 5.2 簡報設計

- 每頁一個重點
- 表格 > 文字
- 流程圖展示施工階段
- 雙欄對比（安全 vs 品質）
- 實績用照片說話
- 最後一頁放衛星標註圖（展示踏勘能力）

### 5.3 答詢準備

- 計畫主持人必須親自答
- 統問統答（3 分鐘準備，10-15 分鐘回答）
- 回答格式：結論 → 理由 → 具體做法
- 不知道的坦承「會後補充」，不要硬掰

## 七、檢查清單

### 投標前最終檢查

- [ ] 服務建議書 ≤ 80 頁
- [ ] 份數 12 份 + 電子檔 1 份
- [ ] 封面有廠商名稱 + 負責人蓋章
- [ ] 內容摘要暨頁次對照表已填
- [ ] 每章頁碼與對照表一致
- [ ] 報價金額與標單一致
- [ ] 所有人員證照影本附件齊全
- [ ] 勞保/健保在職證明附件齊全
- [ ] 簡報資料 12 份已裝入標封
- [ ] 圖說與標單對照表已附

### 內容品質檢查

- [ ] 第二章佔最多頁數（≥ 35%）
- [ ] 每個工法說明有圖說/標單引用標注
- [ ] 施工困難有具體對策（不是空話）
- [ ] 品質管理有具體檢查項目清單
- [ ] 安全管理有校園特殊措施
- [ ] 實績表有金額、期程、業主
- [ ] 零職災/零逾期有明確聲明
